# React Portfolio Website

This is my personal portfolio website built using React and Vite. It showcases my skills, projects, resume, and contact details.

## Features

- Responsive design
- Separate pages using React Router
- About Me section
- Projects section
- Skills section
- Resume download
- Contact section
- Production build using Vite
- Deployed on Vercel

## Technologies Used

- React.js
- Vite
- JavaScript
- HTML5
- CSS3
- React Router
- Git
- GitHub
- Vercel
